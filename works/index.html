📚 学习项目

🧩 Modular RAG MCP Server

简介：一个可插拔、可观测的模块化 RAG（检索增强生成）服务框架，通过 MCP（Model Context Protocol） 协议对外暴露工具接口，可被 Claude Desktop / GitHub Copilot 等 AI 助手直接调用。

这是一个我正在学习与复现的优秀开源项目，原作者 @jerry-ai-dev。我通过研读其架构、跟随 DEV_SPEC 复现代码，系统学习现代 RAG 工程化思路。

技术栈：Python 3.10+ · ChromaDB · LangChain · MCP · Jieba · Streamlit · Ragas

核心能力

  • 🔌 全链路可插拔架构：LLM / Embedding / Reranker / Splitter / VectorStore / Evaluator 全部抽象接口化，配置文件一键切换后端
  • 🔍 混合检索 + 重排：BM25 稀疏检索 + Dense Embedding 向量检索，RRF 融合后接 Cross-Encoder / LLM 精排
  • 🖼️ 多模态图像处理：Vision LLM 自动生成图片描述并缝合进 Chunk，复用纯文本 RAG 链路实现”搜文字出图”
  • 📡 MCP 生态集成：标准协议直接对接 Claude Desktop / Copilot 等客户端，零前端开发
  • 📊 Streamlit 六页面 Dashboard：系统总览 / 数据浏览 / Ingestion 管理 / 链路追踪 / 评估面板
  • 🧪 三层测试体系：Unit / Integration / E2E 全覆盖
  • 🤖 Skill 驱动全流程：auto-coder / qa-tester / package / setup 等 Agent Skill 覆盖完整开发生命周期

我的学习收获

通过这个项目我深入理解了：用 DEV_SPEC 驱动开发的工程化思路，让 AI 基于规格自动完成代码；可插拔架构如何在保证灵活性的同时不牺牲性能；以及如何用 Ragas 等评估框架建立”凭数据迭代”而非”凭感觉调优”的反馈回路。
